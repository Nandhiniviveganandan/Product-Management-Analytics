📊 Product Management Analytics

## Project Overview
End-to-end Data Analytics project
on Product Management using
Python and Power BI.

*Dataset*: 20,000 records | 36 columns
*Tools*: Python | Power BI | ML


## Tools Used
| Tool | Purpose |
|------|---------|
| Python | Data Analysis |
| Pandas | Data Cleaning |
| Matplotlib | Visualization |
| Seaborn | Charts |
| Scikit-learn | ML Models |
| Power BI | Dashboard |


## Project Pipeline
1.  Dataset Creation (20,000 rows)
2.  Data Cleaning
   - Missing values handled
   - Duplicates removed
   - Outliers removed (IQR)
3.  EDA 
   - Revenue Analysis
   - Customer Segments
   - Churn Analysis
   - A/B Testing
4.  Machine Learning
   - Churn Prediction
   - 4 Models trained
5.  Power BI Dashboard
   - 4 Interactive pages


## ML Results
| Model | Accuracy |
|-------|---------|
| Gradient Boosting  | 51.18% |
| Logistic Regression | 50.62% |
| Random Forest | 49.82% |
| Decision Tree | 48.49% |


## Key Insights
- West region → Highest Revenue
- Q4 → Peak Sales Season
- Electronics → Top Category
- 50% users → Churn Risk
-Gradient Boosting → Best Model
